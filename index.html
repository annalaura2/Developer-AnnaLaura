Oi, tudo bem? Meu nome é Anna Laura

Sou Anna Laura, desenvolvedora Full-Stack no sexto semestre de Ciência da Computação. Ao longo da minha trajetória, adquiri experiência no desenvolvimento de landing pages, plataformas de e-commerce, integrações de APIs e sistemas personalizados, utilizando tecnologias como ReactJS, Vue.js, Next.js, TypeScript, Node.js, Java, além de bancos de dados como PostgreSQL, MySQL e MongoDB. Já atuei na Ooka Tecnologia, onde fui responsável pela criação de soluções digitais que otimizam processos e melhoram a experiência do usuário. Minhas responsabilidades incluíam a automação de campanhas de marketing, integração com ferramentas de analytics e CRM, e o desenvolvimento de páginas otimizadas para SEO e conversão. Antes disso, na FATEAM, atuei no desenvolvimento de interfaces UX/UI intuitivas, sempre focada na harmonia entre design e funcionalidade.

Após experiências nas áreas jurídica e administrativa, encontro-me em transição de carreira onde tenho como foco desenvolver minhas qualificações no Desenvolvimento FullStack, criando interfaces intuitivas e estruturadas. Penso que a programação pode transformar positivamente a vida das pessoas, gerando mais acessibilidade e satisfação ao realizar taréfas diarias com mais facilidade, através do aprimoramento ou até mesmo a criação de novas funcionalidades responsivas e visualmente agradáveis, que agreguem no cotidiano dos usuários.

Além das minhas habilidades técnicas, sou uma pessoa que valoriza o trabalho em equipe, sou comunicativa e resiliente, com forte capacidade de adaptação e organização. Busco sempre a melhoria contínua e a colaboração para alcançar soluções eficientes e impactantes.

MINHAS PRINCIPAIS HABILIDADES.

Java

O Java é uma linguagem versátil que permite o desenvolvimento de aplicações para diversas plataformas, oferecendo segurança e alto desempenho. Com sua máquina virtual, garante portabilidade e eficiência, sendo amplamente utilizado em sistemas corporativos, aplicativos móveis, serviços web e diversas outras soluções tecnológicas modernas.

Angular

O Angular é um framework robusto para a construção de aplicações web dinâmicas, permitindo a criação de interfaces interativas e modulares. Com sua arquitetura baseada em componentes, facilita o desenvolvimento escalável e mantém a performance otimizada, garantindo uma experiência fluida e adaptável para diferentes plataformas e dispositivos.

JavaScript

O JavaScript é uma linguagem essencial para o desenvolvimento web, permitindo a criação de sites interativos e dinâmicos. Com o suporte a frameworks e bibliotecas, possibilita a construção de aplicações robustas e escaláveis, sendo executado tanto no front-end quanto no back-end para experiências fluidas e funcionais.

TypeScript

O TypeScript é uma linguagem que adiciona tipagem estática ao JavaScript, proporcionando maior segurança e clareza no código. Com recursos avançados de orientação a objetos, facilita a construção de aplicações escaláveis e robustas, melhorando a produtividade e a manutenção em projetos complexos e de grande porte.

Node.js

O Node.js é um ambiente de execução para JavaScript no servidor, permitindo a criação de aplicações rápidas e escaláveis. Com sua arquitetura orientada a eventos e non-blocking I/O, é ideal para sistemas de alto desempenho, como APIs e plataformas em tempo real, proporcionando eficiência no processamento de grandes volumes de dados.

React

O React é uma biblioteca para construção de interfaces de usuário, focada em componentes reutilizáveis e atualizações rápidas. Com sua abordagem declarativa e virtual DOM, permite criar aplicações dinâmicas e de alta performance, oferecendo uma experiência fluida e responsiva tanto no desenvolvimento quanto no uso.
